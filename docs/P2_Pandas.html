def labelMax(ax, x_offset=-0.02, y_offset = 0.5):
    for p in ax.patches:
        b = p.get_bbox()
        val = "{:.2f}".format(b.y1 + b.y0)        
        ax.annotate(val, ((b.x0 + b.x1)/2 + x_offset, b.y1 + y_offset))

Through our charts we can see that ages of each sex differs greatly by class.  Breaking those classifications down one step further we can find the averages by when the passenger boarded which gives us even more detail.  Using the embarkation point as a split helps us to get a better average of missing ages.

## Fill in empty ages with averages by sex and class and point of embarkation
df['Age_C'] = df['Age'].groupby([df['Sex'], df['Pclass'], df['Embarked']]).apply(lambda x: x.fillna(x.median()))
df['Age_C'] = df['Age_C'].astype(int)

## Fare, lets round these fares to whole dollars for ease of use
df['Fare'] = df['Fare'].astype(int)
